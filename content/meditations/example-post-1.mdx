## Writing Your Post

Use Markdown to format your content. Here are some examples:

### Headings

Use `##` for main sections (not `#` - that's reserved for the title from frontmatter).

### Formatting

You can use **bold text**, *italic text*, and [links to other pages](/about).

### Lists

Create bulleted lists:
- Point one
- Point two
- Point three

Or numbered lists:
1. First item
2. Second item
3. Third item

## Adding Images

Place images in `/public/images/meditations/` and reference them like this:

```markdown
![Description of image](/images/meditations/your-image.jpg)
```

## Frontmatter Fields

- **title**: The post title (required)
- **date**: Publication date in YYYY-MM-DD format (required)
- **excerpt**: Short description for previews (required)
- **image**: Path to header image (optional)
- **tags**: Array of tags for categorization (optional)
- **popularity**: Numeric value for manual sorting, higher = more important (optional)

## Tips for Great Content

- Break up long paragraphs (3-4 sentences max)
- Use headings to organize your thoughts
- Keep excerpts concise and engaging
- Use consistent date formatting
- Link to related posts when relevant
